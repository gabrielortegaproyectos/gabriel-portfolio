# Project Summary
The project is developing a dynamic digital portfolio for Gabriel Ortega Hernández, a Machine Learning Engineer specializing in Educational Assessment. This portfolio effectively showcases his skills, experiences, and publications through an engaging multi-page web application, utilizing modern web technologies to enhance user interaction and visual appeal.

# Project Module Description
The portfolio includes several functional modules:
- **Navbar**: A responsive navigation bar linking to Home, Resume, Portfolio, and Blog.
- **Home Page**: Features an immersive hero section and a "Scrollytelling" effect that presents Gabriel's professional journey, including an interactive visualization of a neural network.
- **Resume Page**: Contains a **Career Timeline** visualization highlighting long-term engagements with interactive tooltips for detailed insights.
- **Portfolio Page**: Showcases interactive demos and project highlights.
- **Blog Page**: A collection of academic publications and articles.
- **Interactive Demo**: Visualizations demonstrating model training metrics.

# File Description Inventory
- **README.md**: Overview of the project and setup instructions.
- **index.html**: The main HTML file for the application.
- **package.json**: Lists project dependencies and scripts.
- **src/**: Contains all source code, including components, pages, and data.
- **public/**: Hosts static assets like images and icons.
- **tailwind.config.ts**: Configuration file for Tailwind CSS.
- **vite.config.ts**: Configuration file for the Vite build tool.

# Technology Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn-ui
- **Data Visualization**: Recharts
- **State Management**: React Hooks
- **Routing**: React Router
- **Build Tool**: Vite
- **Animation**: framer-motion

# Usage
1. **Install Dependencies**: Run `pnpm install` to install all project dependencies.
2. **Build the Project**: Use `pnpm run build` to build the project for production.
3. **Run the Development Server**: Start the local development server with `pnpm run dev`.
